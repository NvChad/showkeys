# Showkeys (WIP)

Eye-candy keys screencaster for Neovim

## Install

```lua
{
    "nvchad/showkeys",
    dependencies = { "NvChad/volt" },
    lazy = true
}
```

## Usage

```
:ShowkeysToggle
```

```lua
  require("showkeys").toggle()
```
